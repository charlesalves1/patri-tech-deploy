Para fazer o push para o GitHub, execute:

git push origin main

Se pedir autenticação, você tem algumas opções:

OPÇÃO 1: Usar Personal Access Token (Recomendado)
--------------------------------------------------
1. Vá em: https://github.com/settings/tokens
2. Gere um novo token com permissões 'repo'
3. Use o token como senha quando o Git pedir

OPÇÃO 2: Configurar SSH
------------------------
1. Gere uma chave SSH: ssh-keygen -t ed25519 -C "seu-email@example.com"
2. Adicione em: https://github.com/settings/keys
3. Altere o remote para SSH: 
   git remote set-url origin git@github.com:charlesalves1/patri-tech.git
4. Faça push: git push origin main

OPÇÃO 3: GitHub CLI
--------------------
1. Instale: https://cli.github.com/
2. Autentique: gh auth login
3. Faça push: git push origin main
